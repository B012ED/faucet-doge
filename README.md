This is a doge generating application using a php program with faucetpay payments

**how to install and usage:**

**Termux:**
* `pkg install php`
* `pkg install git`
* `git clone https://github.com/B012ED/faucet-doge.git`
* `cd faucet-doge`
* `php bot.php`

**Linux:**
* `apt-get install php`
* `apt-get install git`
* `git clone https://github.com/B012ED/faucet-doge.git`
* `cd faucet-doge`
* `php bot.php`
